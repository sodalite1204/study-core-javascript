- **기본형 (primitive type)**
  값이 담긴 주솟값을 바로 복제, 불변성

  - number
  - string
  - boolean
  - null
  - undefined
  - Symbol (+ES6)

- **참조형 (reference type)**
  값이 담긴 주솟값들로 이루어진 묶음을 가르키는 주솟값을 복제
  - object
    - Map
    - WeakMap
    - Set
    - WeakSet
  - Array
  - Function
  - Date
  - RegExp
