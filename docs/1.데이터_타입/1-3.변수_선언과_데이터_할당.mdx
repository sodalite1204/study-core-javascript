### 1-3-1 변수 선언

- 변수 영역에 값을 직접 대입하지 않고 한 단계를 거치는 이유(변수와 데이터를 별도의 공간에 저장)
  - 데이터 변환을 자유롭게 할 수 있음
  - 메모리를 더욱 효율적으로 관리하기 위해
  - 만약 미리 확보한 공간내에서 데이터를 변환할 수 있다면?
    - 확보된 공간을 변환된 데이터 크기에 맞게 늘리는 작업이 필요
    - 중간에 있는 데이터의 공간을 늘려야 한다면, 앞 뒤 데이터를 이동하고 주솟값 변경이 필요
    - 결국 컴퓨터가 해야할 연산이 많아짐
