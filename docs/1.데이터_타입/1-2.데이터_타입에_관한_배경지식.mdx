### 1-2-1 메모리와 데이터

- 메모리: 수많은 비트(하나의 메모리 조각, 고유한 식별자를 통해 위치 확인)로 구성
- JS
  - 숫자→ 정수, 부동소수 구분하지 않고 8바이트(64비트) 확보
- 모든 데이터는 메모리 주솟값 (바이트 단위의 식별자)를 통해 서로 구분하고 연결가능

### 1-2-2 식별자와 변수

- 변수(variable) : 변할 수 있는 값(데이터), 변경 가능한 데이터가 담기는 공간
- 식별자(identifer) : 데이터를 식별하는 데 사용하는 이름
